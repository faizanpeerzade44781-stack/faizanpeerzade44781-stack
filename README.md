### 👋 About Me

I'm a **Data Analyst** based in Bangalore who converts messy, real-world data into KPIs, dashboards, and decisions leadership teams can act on. My day-to-day toolkit is **SQL, Python (Pandas/NumPy), Power BI, and Excel**, and I care as much about the *business story* behind a chart as I do about the query that produced it.

Every project in this profile follows the same discipline: **define the business problem → clean and validate the data → build the metrics → visualize → recommend an action.** No project ships without a written recommendation, because a dashboard without a decision attached to it isn't finished.

- 🔭 Currently deepening my SQL performance-tuning and DAX skills
- 🌱 Learning: Python automation for recurring reporting (replacing manual Excel refreshes)
- 💼 Open to: **Data Analyst / Business Intelligence Analyst** roles (2 YOE)
- 📫 Reach me: **faizanpeerzade44781@gmail.com** · [LinkedIn](https://www.linkedin.com/in/sayed-peerzade-82377a24a)
- ⚡ Fun fact: I treat every dataset like a stakeholder is going to ask "so what?" — because they always do.
